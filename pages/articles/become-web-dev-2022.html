How to become a Web Developer 2022

Becoming a web developer is actually quite simple, you just need to pick your field!

Picking the right tools and frameworks

You should start with NodeJS and Javascript first, then learn Html & CSS

Once you get the hang of it you should learn a framework like ReactJS or VueJS

If you don't like javascript you should learn Python! Frameworks such as Flask and Django are great for web dev(but only for backend)
